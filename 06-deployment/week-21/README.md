# Week 21 — Dashboard & KPI Design

**Phase 6 — Packaging & Deployment &nbsp;|&nbsp; Milestone 6 prep**

---

## This Week's Focus

Plan the dashboard before building it. A good wireframe saves hours of rework.

**Topics:**

- Choose 3–5 KPIs max
- Prioritize clarity over decoration
- Decide page sections: overview, trends, segmentation, insights
- Map each chart to a business/user question
- Define what should be interactive vs static
- Prepare a dashboard wireframe before building

## Task

Draft the dashboard structure:

- Which KPIs will be shown?
- What charts are needed and in what order?
- What data inputs does each section require?
